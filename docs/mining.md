
### Miner Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/tuning-subnet.git
   cd tuning-subnet
   ```

2. Run bootstrap.sh
   ```bash
   sudo -E bash bootstrap.sh
   ```

3. Set up your Hugging Face token:
   ```bash
   export HUGGINGFACE_TOKEN=<your_huggingface_token>
   ```

4. Run the miner:
   ```bash
   task m1_dev
   ```
